You are tasked with building a demo application to showcase the capabilities of Stripe payments for a new direct-to-consumer e-commerce brand called "Cozy Threads". Cozy Threads sells high-quality, ethically-sourced apparel and accessories.

For this take-home project, please build a basic ecommerce web application from scratch with the following requirements:

Product catalog—display a list of Cozy Threads' products, including an image, title, description, and price for each item.

Shopping cart—allow customers to add products to a shopping cart and view the items they've selected.

Checkout—implement a checkout flow that uses the Stripe Payment Element  to securely collect payment information and complete the transaction. Display the result to the customer. 

Write-up—provide a write-up (max 700 words) that covers the following:

The overall architecture and design of your application
Key decisions you made in your implementation, including any trade offs or areas you would improve upon with more time
Any additional features or functionality you would add to the application given more time
 
Evaluation criteria
In this take-home project, we'll be evaluating your ability to:

Design and build a functional ecommerce application
Understand and integrate with Stripe's Payments API
Communicate your technical decisions and trade-offs in a written format
 
Submission requirements
When you're done, please provide the following:

A link to the hosted application
A zipped file or link to the source code of your project
A PDF document with your write-up (max 700 words)
 
Resources / Tips
Leverage existing code samples—To help you build the application, create your own Stripe account and refer to Stripe's developer documentation. The documentation provides detailed information on Stripe's API, SDKs, and integration guides.
Pick a simple framework—We'll ask you to extend your application during the onsite interview, so please structure your code in a simple way that allows for easy modification and addition of new features.
